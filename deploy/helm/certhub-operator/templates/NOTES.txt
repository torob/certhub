spec.secretName. Users who can create or update CerthubCertificate resources
are trusted to choose that Secret name.

The Application token is injected from Secret
{{ .Values.certhub.tokenSecretName }} key {{ .Values.certhub.tokenSecretKey }}
in release namespace {{ .Release.Namespace }}. Restart the operator pod after
rotating it.

The ServiceAccount can create and manage Secrets in its watch scope.

Before upgrading a release that changes the CerthubCertificate schema, apply
the new CRD explicitly. Helm does not upgrade CRDs stored under crds/.
